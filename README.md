# Portfolio Site

Simple static portfolio for Dileepa Dampriya.

## Files

- `index.html` - page structure and content
- `styles.css` - layout, colors, typography, and responsive design
- `script.js` - small scroll reveal animation and footer year

## Quick edits

- Update text sections directly in `index.html`
- Add or remove project cards in the `Selected Work` section
- Change colors and spacing from the `:root` variables in `styles.css`

## GitHub Pages Deployment

This project is ready for GitHub Pages through the workflow at:

- `.github/workflows/deploy-pages.yml`

After the repository is pushed to GitHub:

1. Create a new GitHub repository.
2. Push this project to the `main` branch.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `GitHub Actions`.
5. Push changes to `main` and GitHub will publish the site.
